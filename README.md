# Web-development-E-commerce-web-app
Develop an e-commerce web application using recent technologies such as NextjS, ReactJS, ASP.NET, NodeJs, Strapi, or any recent technologies of your choice. (Flutter + Dart)

# Weekly Report Added (Subject to changes)
Week 2 tasks: Create project folders and assign team tasks.\
Week 3 tasks: Basic website template, navigation, and logo. (Home, Products, Cart,
Checkout)\
Week 4 tasks: Advance website layout, implement CSS, put content on existing
pages.\
Week 5 tasks: Create product data (Prices, descriptions, images, and display)\
Week 6 tasks: Implement add to cart functionality\
Week 7 tasks: Implement shopping cart functionality (quantity, price, total cost)\
Week 8 tasks: Increase shopping cart functionality (remove item, number of items,
shipping)\
Week 9 tasks: Create checkout page with data validation\
Week 10 tasks: Order confirmation page and functionality (clearing cart,
confirmation of order)\
Week 11 tasks: Admin tools (Creating new items, login requirements?)\
Week 12 tasks: Further advance admin tools (Delete items, edit items)\
Week 13 tasks: Website testing (finding bugs, user testing)\
Week 15 tasks: Project Presentation, Poster board or live demonstration
